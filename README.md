## 💻 Sobre o Projeto
Este projeto foi desenvolvido para oferecer uma solução simples, rápida e intuitiva para usuários que precisam converter valores 
entre diferentes moedas (como Real, Dólar e Euro) sem complicações. 

O foco principal foi criar uma interface limpa e aplicar conceitos fundamentais de programação para garantir um cálculo preciso 
e em tempo real.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica da página e dos campos de entrada.
- **CSS3:** Estilização da interface, garantindo um design moderno, limpo e responsivo.
- **JavaScript (ES6):** Toda a inteligência da aplicação, controle de fluxo e cálculos.

## 🧠 O que eu pratiquei e aprendi neste projeto:
- **Manipulação de DOM:** Captura de valores de campos de texto (`input`) e listas de seleção (`select`) usando JavaScript.
- **Lógica Condicional:** Uso de estruturas condicionais para identificar quais moedas foram selecionadas para aplicar a taxa de conversão correta.
- **Eventos de Interface:** Uso de `addEventListener` para disparar o cálculo automaticamente assim que o usuário interage com o botão.
- **Tratamento de Dados:** Formatação de números para exibição de casas decimais monetárias de forma correta.
